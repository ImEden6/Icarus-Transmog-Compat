# Icarus-Transmog Compat

A compatibility mod that allows Icarus wings to render with appearances from the Transmog mod.

## Description

This mod bridges the gap between [Icarus](https://www.curseforge.com/minecraft/mc-mods/icarus) and [Transmog](https://www.curseforge.com/minecraft/mc-mods/transmog).
When you apply a transmog appearance to your Icarus wings item, they will render in-game using the transmogged model and texture instead of the default Icarus appearance.

## Dependencies

*   Fabric Loader
*   Fabric API
*   Icarus
*   Transmog

## Installation

1.  Download the latest release.
2.  Place the `.jar` file in your `mods` folder.
3.  Ensure you have all dependencies installed.
